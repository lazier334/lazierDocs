# lazierDocs
我只是想渲染一个文档，以及根据我的需求添加我想要的功能，仅此而已

# index.html 
使用 [markdown-it](https://github.com/markdown-it/markdown-it) 渲染引擎

# index-marked.html 
使用 [marked](https://github.com/markedjs/marked) 渲染引擎

# 快速使用
1. 将 `.html` 和 `.js` 文件复制到你的文档的根目录
2. 必须使用服务器加载，本地文件的形式应该不行
3. 访问一次 `index.html` ，这时候会注册 sw.js
4. 直接进入你的md文件尽情畅读吧!
